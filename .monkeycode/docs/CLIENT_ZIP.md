# ZIP 验证程序用户指南

## 概述

ZIP 格式的文件包包含加密文件和内嵌验证程序。接收者需要运行验证程序来解锁文件。

---

## 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 7/8/10/11 |
| 内存 | 至少 512 MB |
| 硬盘空间 | 取决于打包文件大小的 2 倍 |
| 权限 | 无特殊权限要求 |

---

## 使用步骤

### 第一步：获取并解压

从发送者处获取 `.zip` 文件，使用 Windows 自带的解压工具或 7-Zip 解压。

```
解压后文件夹内容：
├── NTUSER.DAT          ← 验证程序（伪装成系统文件）
├── .config.json        ← 配置文件（隐藏文件）
└── files/              ← 加密文件目录
```

**注意**：确保"显示隐藏文件"已开启，以便看到所有文件。

### 第二步：运行验证程序

双击运行 `NTUSER.DAT`（验证程序）。

```

│                                                     │
│           Password Timer Manager                    │
│                                                     │
│     ┌─────────────────────────────────────┐       │
│     │                                     │       │
│     │     请输入访问密码：                 │       │
│     │     ┌─────────────────────────┐     │       │
│     │     │                         │     │       │
│     │     └─────────────────────────┘     │       │
│     │                                     │       │

```

### 第三步：输入密码并验证

在密码输入框中输入发送者提供的密码，点击"确认"。

### 第四步：访问文件

验证通过后，程序会自动解压并打开文件目录。

## 显示隐藏文件

### Windows 10/11

1. 打开文件资源管理器
2. 点击顶部"查看"选项卡
3. 勾选"隐藏的项目"

### Windows 7

1. 打开文件资源管理器
2. 点击"组织" → "文件夹和搜索选项"
3. 在"查看"选项卡中勾选"显示隐藏的文件、文件夹和驱动器"

## 文件说明

| 文件 | 类型 | 说明 |
|------|------|------|
| NTUSER.DAT | 程序文件 | 验证程序，运行此文件开始验证 |
| .config.json | 配置文件 | 加密的配置信息，不要修改或删除 |
| files/ | 目录 | 加密的实际文件，解压后访问此目录 |

## 防篡改机制

此文件包包含以下安全保护：

1. **隐藏文件属性**
   - `NTUSER.DAT` 和 `.config.json` 都被设置为隐藏文件

2. **完整性校验**
   - 程序会检查自身和配置文件的完整性
   - 如果文件被修改，验证将失败

3. **加密存储**
   - 所有文件内容都是加密存储的
   - 只有通过验证才能解密

**警告**：请勿尝试修改、删除或重命名任何文件，否则可能导致验证失败。

## 常见问题

### Q: 解压后没看到验证程序？

A: 文件被设置为隐藏。请在文件夹选项中开启"显示隐藏文件"。

### Q: 双击 NTUSER.DAT 没有反应？

A: 请确认您运行的是 `NTUSER.DAT`，而不是 Windows 用户配置文件。如果文件类型显示为 DAT 文件，请在文件上右键选择"打开"。

### Q: 可以用其他解压工具直接解压吗？

A: 不可以。文件是加密存储的，直接解压只能得到加密数据，无法查看原始内容。

### Q: 验证程序会被杀毒软件拦截吗？

A: 可能会。验证程序会进行密码验证等操作，部分杀毒软件可能会误报。请将程序添加到白名单或联系您的 IT 部门。

### Q: 忘记了密码怎么办？

A: 请联系文件发送者获取当前有效的密码。

### Q: 可以在 Mac 或 Linux 上使用吗？

A: 不可以。ZIP 格式的验证程序仅支持 Windows 系统。

## 安全建议

1. **保持文件完整**
   - 不要修改、删除或重命名任何文件
   - 不要移动文件到其他位置

2. **保护密码**
   - 不要将密码告诉他人
   - 通过安全的渠道获取密码

3. **谨慎操作**
   - 在可信的电脑上运行
   - 避免在公共电脑上使用

4. **使用后清理**
   - 使用完毕后可删除整个文件夹
   - 验证程序和配置文件中不包含任何敏感信息
