## 核心组件

### 1. API 服务

| 组件 | 说明 | 技术栈 |
|------|------|--------|
| HTTP Server | 处理 HTTP 请求 | Gin/FastAPI/Spring Boot |
| Router | 路由分发 | 根据语言框架 |
| Middleware | 认证、限流、日志 | 根据语言框架 |
| Service | 业务逻辑 | 业务代码 |
| Repository | 数据访问 | SQL/ORM |

### 2. 前端服务

| 组件 | 说明 | 技术栈 |
|------|------|--------|
| SPA | 单页应用 | React 18+ / Vue 3+ |
| State Management | 状态管理 | Zustand/Pinia |
| HTTP Client | API 调用 | Axios |
| UI Framework | UI 组件 | Ant Design / Element Plus |

### 3. 客户端验证程序

| 组件 | 说明 |
|------|------|
| Config Loader | 加载加密配置 |
| Signature Verifier | RSA 签名验证 |
| Crypto Engine | AES 解密 |
| Password Validator | 密码验证 |
| File Extractor | 文件解压 |

## 数据流

### 创建文件包流程

```
用户上传文件
    │
    ▼
┌─────────────────────────────────────┐
│  1. Web 前端 / API 接收文件         │
│     - multipart/form-data           │
│     - 文件大小检查                   │
│     - 文件类型检查                   │
└─────────────────┬───────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│  2. 保存原始文件                     │
│     - 生成 UUID                      │
│     - 存储到文件系统/S3              │
│     - 计算文件哈希                    │
└─────────────────┬───────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│  3. 创建密码策略                     │
│     - 密码加密存储                   │
│     - 设置优先级和有效期             │
└─────────────────┬───────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│  4. 生成下载包                       │
│     - 调用 Generator 服务           │
│     - 生成 EXE 或 ZIP                │
│     - 嵌入加密配置                   │
│     - 签名验证                       │
└─────────────────┬───────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│  5. 返回下载链接                     │
│     - 保存包信息到数据库             │
│     - 生成下载 Token                 │
└─────────────────────────────────────┘
```

### 密码验证流程

```
用户输入密码
    │
    ▼
┌─────────────────────────────────────┐
│  1. 检查配置文件完整性               │
│     - 文件存在检查                   │
│     - 哈希校验                       │
└─────────────────┬───────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│  2. 在线验证（优先）                 │
│     - POST /api/v1/verify           │
│     - 检查密码是否有效               │
│     - 返回解密密钥                   │
│                                     │
│  ⚠️ 网络异常？                       │
│     ↓ 是                           │
│  ┌─────────────────────────────────┐│
│  │ 3. 离线验证（兜底）              ││
│  │    - RSA 签名验证                ││
│  │    - 解密内置配置                ││
│  │    - 密码哈希比对                ││
│  │    - 获取解密密钥                ││
│  └─────────────────────────────────┘│
└─────────────────┬───────────────────┘
                  │
                  ▼
┌─────────────────────────────────────┐
│  4. 解压文件                         │
│     - 使用密钥解密                   │
│     - 解压到临时目录                 │
│     - 打开文件                       │
└─────────────────────────────────────┘
```

### 容灾设计

| 组件 | 容灾策略 |
|------|----------|
| API 服务 | 多实例部署，Kubernetes 自动重启 |
| PostgreSQL | 主从复制，定期备份 |
| Redis | Redis Cluster 或 Sentinel |
| 文件存储 | 多副本存储（S3/OSS） |
| 前端 | CDN 加速 |
