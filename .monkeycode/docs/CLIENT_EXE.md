# 自解压程序用户指南 (EXE)

## 概述

EXE 自解压程序是 Password Timer Manager 生成的单一可执行文件，接收者无需安装任何额外软件即可使用。

---

## 系统要求

| 项目 | 要求 |
|------|------|
| 操作系统 | Windows 7/8/10/11 |
| 内存 | 至少 512 MB |
| 硬盘空间 | 取决于打包文件大小 |
| 权限 | 无特殊权限要求 |

---

## 使用步骤

### 第一步：获取文件

从发送者处获取 `.exe` 文件。

**注意**：首次运行可能会出现 Windows 安全警告，点击"仍要运行"即可。

### 第二步：运行程序

双击 `.exe` 文件启动验证程序。

```
┌─────────────────────────────────────────────────────┐
│                                                     │
│           Password Timer Manager                    │
│                                                     │
│     ┌─────────────────────────────────────┐       │
│     │                                     │       │
│     │     请输入访问密码：                 │       │
│     │     ┌─────────────────────────┐     │       │
│     │     │                         │     │       │
│     │     └─────────────────────────┘     │       │
│     │                                     │       │
│     │         [ 确认 ]  [ 取消 ]          │       │
│     │                                     │       │
│     └─────────────────────────────────────┘       │
│                                                     │
│     当前可用密码有效期：                            │
│     2024-01-01 00:00 ~ 2024-01-07 23:59          │
│                                                     │
└─────────────────────────────────────────────────────┘
```

### 第三步：输入密码

在密码输入框中输入发送者提供的密码。

**提示**：
- 密码区分大小写
- 输入框支持粘贴功能

### 第四步：验证并解压

点击"确认"后，程序将验证密码：

- **密码正确且有效**：自动解压文件到当前目录或指定目录
- **密码错误**：显示错误提示，可重新输入
- **密码已过期**：显示过期提示，联系发送者获取新密码

```
验证成功！
正在解压文件...
解压完成！
```

### 第五步：访问文件

解压完成后，程序会自动打开包含文件的文件夹，或者显示文件列表供您选择。

## 高级功能

### 解压到指定目录

在主界面点击"浏览"可以选择解压目录。

### 查看文件信息

点击"文件信息"可以查看：
- 原始文件名
- 文件大小
- 打包时间
- 发送者备注

### 离线使用

程序支持离线验证。当网络不可用时，程序会自动使用内置的密码策略进行验证。

## 常见问题

### Q: 为什么程序被 Windows 拦截？

A: 首次运行自解压程序时，Windows SmartScreen 可能会拦截。点击"更多信息"→"仍要运行"即可。

### Q: 密码输入错误有次数限制吗？

A: 在线模式下没有限制。离线模式下连续 5 次错误后会暂时锁定 30 分钟。

### Q: 解压失败怎么办？

A: 请确认：
1. 硬盘空间是否充足
2. 当前目录是否有写入权限
3. 尝试解压到其他位置

### Q: 如何确认文件来源可靠？

A: 请确认：
1. 文件来自您认识的可信联系人
2. 密码是通过安全方式获取的
3. 程序的数字签名是否有效

---

## 安全说明

1. **不要**将密码告诉他人
2. **不要**在不受信任的电脑上输入密码
3. 解压后的文件操作与普通文件相同
4. 程序不会在后台运行或收集信息
